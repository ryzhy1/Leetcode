Постройте поле для игры "Сапёр" (почитать про игру) по следующему алгоритму:

Пользователь выбирает размеры поля.
Пользователь выбирает количество мин.
Пользователь вводит координаты первой выбранной клетки поля.
Программа выполняет заполнение поля:

Случайным образом на поле наносится введенное количество мин (кроме клетки, которую пользователь отметил первой)
Для каждой пустой клетки рассчитывается число – количество мин в восьми клетках, окружающих текущую. Если клетка расположена в углу или у бортика поля, то количество окружающих клеток уменьшается соответственно. Например, для угловой клетки их будет 3.
Решение оформите в виде одной или нескольких функций с типизацией входных и выходных значений.

Для демонстрации решения выведите поле в виде кортежа кортежей.
